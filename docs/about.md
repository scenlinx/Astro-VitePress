# 关于 VitePress

## 项目背景

VitePress 是一个由 Vue 团队开发的静态网站生成器，专为构建快速、现代化的文档网站而设计。

## 核心理念

- **速度优先**: 基于 Vite 构建，提供极速的开发体验
- **内容为王**: Markdown 优先，让作者专注于内容创作
- **渐进增强**: 从简单的文档到复杂的自定义主题，灵活扩展

## 技术栈

- **构建工具**: Vite
- **UI 框架**: Vue 3
- **内容格式**: Markdown / MDX
- **输出**: 静态 HTML

## 开源

VitePress 是完全开源的项目，采用 MIT 许可证。
