# 关于 Astro-VitePress

## 项目背景

Astro-VitePress 是一个基于 [Astro 5](https://astro.build) 构建的静态文档网站生成器。它在功能和体验上借鉴了 VitePress，但保持极简的依赖设计——仅依赖 Astro 一个包。

## 核心理念

- **速度优先**: 基于 Astro 构建，默认零 JavaScript 输出纯静态 HTML，加载极速
- **内容为王**: Markdown 优先，让作者专注于内容创作
- **零配置**: 将 Markdown 文件放入 `docs/` 目录，自动生成导航、侧边栏、目录和分页
- **渐进增强**: 从简单文档到复杂的自定义主题，灵活扩展

## 技术栈

- **框架**: Astro 5
- **语言**: TypeScript
- **样式**: 纯 CSS（CSS 自定义属性实现主题）
- **语法高亮**: Shiki
- **搜索**: 客户端全文搜索
- **输出**: 纯静态 HTML

## 开源

Astro-VitePress 是完全开源的项目，采用 MIT 许可证。源码托管在 [GitHub](https://github.com/scenlinx/astro-vitepress)。
