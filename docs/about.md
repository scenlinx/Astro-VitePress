# 关于 astro-vitepress

## 项目背景

astro-vitepress 是一个基于 Astro 5 构建的静态文档网站生成器，灵感来源于 VitePress。它使用 Markdown 文件作为内容源，自动生成导航、侧边栏、目录和分页，提供类似 VitePress 的文档体验。

## 核心理念

- **速度优先**: 基于 Astro 构建，输出纯静态 HTML，加载极速
- **内容为王**: Markdown 优先，让作者专注于内容创作
- **零配置**: 将 Markdown 文件放入 `docs/` 目录，自动生成完整站点
- **渐进增强**: 从简单的文档到复杂的自定义主题，灵活扩展

## 技术栈

- **框架**: Astro 5
- **内容格式**: Markdown
- **输出**: 静态 HTML

## 开源

astro-vitepress 是完全开源的项目，采用 MIT 许可证。源码托管在 [GitHub](https://github.com/scenlinx/astro-vitepress)。
